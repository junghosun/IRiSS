# IRiSS

[Legislative Co-Sponsorship.R](<Legislative Co-Sponsorship.R>) is a R file which I coded for one of my working papers. Using data from South Korean legislature, this file calculates network distance, W-NOMINATE scores, and conducts empirical analyses as shown in the code and working paper.

[Poster Processing.py](<Poster Processing.py>) is a Python file which extracts three dominant colors from an election poster, and calculates its CIE Delta E 2000 distance with the party's official color to conduct analysis.
