# IRiSS

Legislative Co-Sponsorship.R is a R file which I coded for one of my working papers. Using data from South Korean legislature, this file calculatues network distance, W-NOMIANTE scores, and conduct empirical analyses as shown in the code and working paper.

Poster Processing.py is a Python file which extracts three dominant colors from an election poster, and calculates its CIE Delta E 2000 distance with the party's official color to conduct analysis.
